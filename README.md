# Job Form Autofill Chrome Extension

## Overview
This Chrome extension helps users quickly autofill common job application fields like name, email, phone number, and skills with one click.

## Tech Stack
- JavaScript (ES6)
- Chrome Extensions (Manifest V3)

## Setup
1. Download the ZIP
2. Extract it
3. Open Chrome and go to chrome://extensions
4. Enable Developer Mode
5. Click Load Unpacked
6. Select the extracted folder

## Usage
1. Open a job application form
2. Click the extension icon
3. Click "Autofill Form"
